To switch the frontend from tobalaba to olichain do the following steps:
-------------------------------------------------------------------------

1. Navigate to the folder olichain (.\olichain)
2. Copy the file oliDashboard.js
3. Navigate one folder back (into the folder js)
4. Paste the the file oliDashboard.js into the js folder. -> That will replae the existing 'oliDashboard.js' file in the js folder.

To switch back (from tobalaba to oliChain) repaete step 1-4. 
Use the 'oliDashboard.js' file that is located in tobalaba instead of olichain
